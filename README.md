ESP32-S2 and SI5351 based VFO. It have WIFI hotspot and http based user interface. 

This code is made for vintage FM tranceiver Salora SRP12/50
Features
set RX and TX frequency separately. 
set CTCSS tone 
Frequency formula can be changed inside code so that http set final frequency and local oscillator
have calculated frequency 

Hardware required.
Some FM radio which have known local oscillator and TX chrystal frequency
ESP32-S2 evalution board
Si5351 evalution board
Some resistors and capasitors etc.
